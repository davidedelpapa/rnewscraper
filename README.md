# rNewscraper
A news scraper written in Rust

#### version = "0.0.1"
For now it handles simply [Hacker News](https://enews.ycombinator.com/)'s homepage; and ain't doing a great job, at it.

## Build
Download:
``` sh
git clone https://github.com/davidedelpapa/rnewscraper.git
```
Build & Run:
``` sh
cargo run
```
